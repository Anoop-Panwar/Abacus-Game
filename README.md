# 🧮 Abacus Learning Game

An interactive **5-Digit Abacus Simulation** built using **HTML, CSS, and Vanilla JavaScript**.

This project generates a random 5-digit number and challenges the user to match it by adjusting the abacus rods.

---

## 🚀 Features

- 🔢 Random 5-digit number generation (00001 – 99999)
- 🧠 Array-based state management
- ➕ Increment with carry handling
- ➖ Decrement functionality
- 🎯 Win condition detection
- 🔄 Reset / New Question option
- 🎨 Clean and simple UI design

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Flexbox)**
- **JavaScript (ES6)**
